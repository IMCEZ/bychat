# Bychat（边聊）

Android 10+ 去中心化聊天应用。任意设备可作为单房间服务器，消息保存在主机本地；客户端可通过 IP 和端口连接，也可接入兼容的外部多房间服务器。支持文字、录音消息、房主踢人、禁言、封禁及清空记录。

## 手机构建 APK

1. 在 GitHub 新建仓库。
2. 将本目录全部文件上传到仓库，默认分支使用 `main`。
3. 打开仓库的 `Actions`，选择“构建边聊APK”并运行。
4. 构建结束后，在任务页面的 Artifacts 下载 `Bychat-debug-apk`，解压并安装 APK。

## 公网连接

主机默认监听 TCP `18888`。主机处于路由器后方时，需要在路由器中把该 TCP 端口转发到手机的局域网 IP；运营商 CGNAT 网络需要公网 IP、IPv6 端口放行或第三方 TCP 隧道。客户端填写主机公网 IP/域名和映射后的端口。

## 外部服务器协议

通信为 UTF-8 JSON Lines。客户端首包 `action=login`，消息包 `action=message`；每次连接可指定 `room`，因此兼容服务器可提供多个房间。内置手机服务器按设计只提供一个房间。
